# 🔐 IoT Cybersecurity - Machine Learning Project

## 📌 Dataset
Edge-IIoTset Cyber Security Dataset of IoT & IIoT  
Source: Kaggle (Mohamed Amine Ferrag)  
- 2.2 Million rows, 39 features  
- 15 attack types (DDoS, SQL Injection, Ransomware etc.)

## 🛠️ Techniques Used
- Data Preprocessing & Cleaning
- PCA (Principal Component Analysis)
- SVD (Singular Value Decomposition)
- KNN (K-Nearest Neighbors)
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest

## 📊 Final Results
| Model | Accuracy |
|---|---|
| KNN | 88.77% |
| Linear Regression | 72.84% |
| Logistic Regression | 88.55% |
| Decision Tree | 89.94% |
| Random Forest | 90.01% |

## ✅ Best Model
**Random Forest — 90.01% Accuracy**
